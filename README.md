# 🍋 Little Lemon — Table Reservation Web App

Capstone project for the **Meta Front-End Developer Professional Certificate** on Coursera.  
A high-performance, responsive web application for the Little Lemon restaurant, featuring an advanced online table reservation system, e-commerce menu, and modern UI/UX.

---

## 🛠️ Features

### Core Features
- **Responsive Table Reservation:** Dynamic booking form with real-time time slot availability based on selected date.
- **E-commerce Menu:** Order online with a shopping cart, automatic calculations (Subtotal, IVA), and a simulated delivery checkout flow.
- **Cart Persistence:** Shopping cart data is persisted using `localStorage`.

### Technical Enhancements
- **Dynamic Content:** Special Offer of the Day showcased on the homepage.
- **Modern Animations:** Smooth page transitions implemented using `framer-motion`.
- **Performance Optimized:** Code splitting applied to all routes using `React.lazy` and `Suspense`.
- **Clean Architecture:** Logic refactored into custom hooks (`useCart`) for better maintainability.
- **Accessibility:** Semantic HTML, ARIA labels, and keyboard-navigable components.
- **Robust UI:** Consistent styling with CSS Grid and Flexbox following brand guidelines.

## 🚀 How to run locally

```bash
# 1. Clone the repository
git clone https://github.com/anudoranador87/little-lemon.git

# 2. Enter the directory
cd little-lemon

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
```
