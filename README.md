## 📋 Descripción del proyecto
Little Lemon es un restaurante mediterráneo que necesitaba una solución para que sus clientes pudieran **reservar mesa online**. Este proyecto cubre todo el proceso: desde el diseño UX/UI en Figma hasta el desarrollo completo con React.

---

## 🛠️ Tecnologías utilizadas
- **React** (create-react-app)
- **JavaScript ES7+**
- **HTML5 / CSS3**
- **Figma** (diseño UX/UI)
- **Git & GitHub** (control de versiones)

---

## 🚀 Cómo ejecutar el proyecto localmente

```bash
# 1. Clonar el repositorio
git clone https://github.com/TU_USUARIO/little-lemon.git

# 2. Entrar a la carpeta
cd little-lemon

# 3. Instalar dependencias
npm install

# 4. Iniciar la app
npm start
```
