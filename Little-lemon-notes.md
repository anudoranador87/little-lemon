## 🏗️ HTML Semántico
 
Tengo que usar etiquetas semánticas, no divs para todo. Mejora la accesibilidad y el SEO.
 
Etiquetas que tengo que usar:
- header — cabecera de la página
- nav — navegación
- main — contenido principal
- section — cada sección de la página
- article — cada plato en destacados
- footer — pie de página
---
 
## ♿ Accesibilidad
 
- Todas las imágenes deben tener texto alternativo (alt) descriptivo
- Si una imagen es decorativa, el alt va vacío
- Cada input del formulario debe tener su label asociado
- Los botones deben tener texto descriptivo
- La app debe poder navegarse con el teclado (Tab)
- El contraste de colores debe ser suficiente para leer bien
---
 
## 📱 Responsive Design
 
- Diseño mobile first — primero diseño para móvil, luego para tablet y escritorio
- Breakpoints a usar: 320px (móvil), 768px (tablet), 1024px (escritorio)
- La sección de destacados pasa de 1 columna en móvil a 3 en escritorio
